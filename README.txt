
BeninMarket - Prototype (static)
--------------------------------

Contenu:
- index.html
- listings.html
- about.html
- css/styles.css
- js/script.js

But:
- C'est une maquette statique (front-end uniquement) pour valider l'UX et présenter l'idée.
- Pour rendre la marketplace pleinement fonctionnelle, il faudra:
  1) Ajouter un backend (ex: Firebase, Supabase, Node.js + PostgreSQL, Laravel).
  2) Gérer l'authentification (email, WhatsApp, Google).
  3) Mettre en place les paiements (Mobile Money, Flooz, cartes - via API).
  4) Ajouter modération d'annonces, upload d'images réelles, recherche avancée, chat interne.

Déploiement rapide:
1) Hébergement statique (gratuit) : GitHub Pages, Netlify, Vercel.
   - Exemple: créer un repo GitHub et pousser ces fichiers, activer Pages.
2) Hébergement payant / professionnel : Hostinger, OVH, ou un VPS.
3) Pour une app mobile, transformer le site en PWA ou utiliser un wrapper (Cordova/Capacitor) ou développer une appli native.

Je peux t'aider ensuite à:
- Mettre en place un backend simple (Supabase ou Firebase) et connecter les formulaires.
- Préparer les scripts de déploiement pour GitHub Pages / Netlify.
- Créer un plan marketing pour les 1ers 1000 utilisateurs.
